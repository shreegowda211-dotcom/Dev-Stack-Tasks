A collection of responsive web applications developed as part of the **DevStack Technologies Web Development Internship** using **HTML5, CSS3, and JavaScript**.

Each project focuses on core web development concepts including DOM manipulation, event handling, responsive design, and JavaScript logic.

---

## 📂 Projects

### 🧮 Project 1 — Simple Calculator

A responsive calculator for performing basic arithmetic operations.

**Features**

* Addition
*  Subtraction
*  Multiplication
*  Division
*  Percentage
*  Keyboard Support
*  Clear & Backspace
*  Responsive Design

---

### 🔐 Project 2 — Password Generator

A customizable password generator with clipboard support.

**Features**

* Adjustable password length
* Uppercase letters
* Lowercase letters
* Numbers
* Symbols
* Copy to Clipboard
* Responsive UI

---

### ⏱️ Project 3 — Stopwatch

A digital stopwatch with lap recording functionality.

**Features**

* Start
* Pause
* Reset
* Lap Recording
* Responsive Design

---

### 🎂 Project 5 — Age Calculator

Calculates a user's exact age from their date of birth.

**Features**

* Date Picker
* Calculates Years, Months & Days
* Input Validation
* Responsive UI

---

### Clone the repository

```bash

```

### Open the project

```bash
cd Dev-Stack-Tasks
```

Open any project folder and launch `index.html` in your browser, or use the **Live Server** extension in Visual Studio Code.
