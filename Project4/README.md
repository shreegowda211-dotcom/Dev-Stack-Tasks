# Age Calculator

A premium age calculator web app with a polished UI and rich age insights.

## Overview

This project calculates age from a selected date of birth and provides extra details such as days lived, next birthday countdown, and a personality-style age category.

## Features

- Clean red/cream/black premium interface
- Accurate age calculation in years, months, and days
- Days lived summary
- Next birthday countdown with friendly messaging
- Age category insight for a professional feel

## Files

- `index.html` — main structure and form markup
- `style.css` — premium visual styling
- `script.js` — age calculation and display logic
- `Project4.mp4` — attached demo video showing the app in action

## Usage

1. Open `index.html` in a browser.
2. Select your date of birth using the date picker.
3. Click **Calculate Age**.
4. View the calculated age summary and countdown details.
