# Stopwatch

A premium ocean-inspired stopwatch app with smart lap tracking and a polished user interface.

## Overview

This stopwatch app offers clean timing controls, lap tracking, and a premium UI design. It calculates elapsed time and saves lap records, showing the best lap and total lap count.

## Features

- Ocean blue and yellow gradient premium UI
- Start, pause, reset, and lap controls
- Real-time stopwatch display with hours, minutes, and seconds
- Logged lap list with lap times
- Best lap detection and total lap counter

## Files

- `index.html` — stopwatch markup and structure
- `style.css` — premium ocean-inspired styling
- `script.js` — stopwatch functionality and lap tracking logic
- `Project3.mp4` — attached demo video showing the app in action

## Usage

1. Open `index.html` in a browser.
2. Start the stopwatch with the **Start** button.
3. Use **Lap** to save lap times and **Pause** to stop the timer temporarily.
4. Reset the stopwatch to clear all lap records.
